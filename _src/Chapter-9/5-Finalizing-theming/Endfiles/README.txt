Paste the Packt folder and contents in the app/code folder of your Magento application. 

Copy the content of https://github.com/kenwheeler/slick/blob/master/slick/slick.css into the file app/code/Packt/ProductSlider/view/frontend/web/css/source/module/_slick.less
Copy the content of https://github.com/kenwheeler/slick/blob/master/slick/slick.js into the file app/code/Packt/ProductSlider/view/frontend/web/js/slick.js

Clean the cache and run the command

php bin/magento setup:upgrade
